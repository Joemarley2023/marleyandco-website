# Marley & Co — One project, entered once, used everywhere

You now have a **single source of truth**. When you add or edit a project in the dashboard (app.pagescms.org → **Projects**), that one entry automatically powers three things — no re-typing.

## The project template (what you fill in)
Each project captures:
- **Project name**, **Location**, **Year**
- **Short description** (1–2 lines)
- **Key materials** (comma separated, e.g. "American oak, marble, stainless steel")
- **Architect**, **Interior designer**, **Builder**, **Photographer**
- **Instagram hook** (optional one-liner; if blank, the description is used)
- **Lead image**, **two detail images**, **gallery photos**

## Where that information goes automatically

**1. Website** (`marleyandco.com.au`)
- The project appears in *Selected Work* with its lead image + gallery.
- The **credit line** builds itself from whichever of Architect / Interiors / Builder / Photographer you filled in (empty ones are simply left off).

**2. Collaborators** (the logo grid)
- Anyone you credit on a project is **added to the Collaborators grid automatically**, sorted into Architects / Designers / Builders / Photographers — no separate data entry.
- Duplicates are merged (credit "Olaver Architecture" on five projects → they appear once).
- The manual Collaborators list is still there for adding a studio you haven't done a project with yet, or for uploading logos.

**3. Instagram captions** (`marleyandco.com.au/studio.html` — internal tool)
- Generates a ready-to-post **caption + hashtags** for every project from the same data.
- Hashtags include a base set plus ones derived from your materials (oak → #americanoak, marble → #marble, etc.).
- Shows you a tick-list of which credits are filled vs missing, and a reminder of who to **@tag**.
- **Copy caption + hashtags** button → paste straight into Instagram.

## So the workflow is simply:
1. New project shot? → open **Projects** in the dashboard → **Add an entry**, fill the template, upload photos → **Save**.
2. The website updates itself (~1–2 min).
3. The collaborators grid updates itself.
4. Open **/studio.html**, hit **Copy** on that project, paste into Instagram.

That's it — enter once, used everywhere.

> Note: `studio.html` is internal (set to "noindex" and not linked from the public site). Bookmark it. To find it, go to `marleyandco.com.au/studio.html` (or the netlify.app address for now).
